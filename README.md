# 🛒 쇼핑 리스트 앱

한국어 쇼핑 리스트 웹 애플리케이션입니다. 순수 HTML/CSS/JavaScript로 만들어졌으며 `localStorage`를 사용해 데이터를 저장합니다.

## 기능

- 아이템 추가 (버튼 클릭 또는 Enter 키)
- 아이템 완료 체크/해제
- 개별 아이템 삭제
- 완료된 항목 일괄 삭제
- 완료 통계 표시 (N / M 완료)
- localStorage를 통한 데이터 영구 저장

## 사용 방법

`shopping-list.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

## 테스트 실행

Playwright를 사용한 자동화 테스트가 포함되어 있습니다.

```bash
npm install
node test-shopping.mjs
```

총 10개의 테스트 시나리오가 포함되어 있습니다:
1. 초기 상태 확인
2. 아이템 추가 (버튼 클릭)
3. 아이템 추가 (Enter 키)
4. 빈 입력 무시
5. 체크 (완료) 기능
6. 체크 해제
7. 아이템 삭제
8. 완료된 항목 일괄 삭제
9. LocalStorage 저장 유지
10. 모두 삭제 후 빈 상태 복귀
